# Multi-Agent Research Assistant

A powerful AI-driven Multi-Agent Research Assistant built using LangChain, LangGraph, and OpenAI. The system employs multiple specialized agents that collaborate to search, scrape, analyze, critique, and generate high-quality research reports on any topic. It integrates web search and web scraping tools to gather real-time information, processes the data through an LCEL-based pipeline, and produces structured, reliable, and comprehensive outputs through coordinated agent workflows.

### Features

* Multi-agent architecture for specialized task handling
* Web search and web scraping capabilities
* Automated research and information gathering
* Report generation and content refinement
* LangChain and LangGraph powered workflows
* Streamlit-based interactive user interface
* Modular and scalable design for future enhancements

### Tech Stack

* Python
* LangChain
* LangGraph
* OpenAI API
* BeautifulSoup4
* Streamlit
* Requests
* LCEL (LangChain Expression Language)
